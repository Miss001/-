# 社区版
## OBD
集群管理工具，用于集群的安装、部署、配置、启动、停止、升级等操作。     
(与tidb的tiup功能有点类似)
- 查看现有集群
```
obd cluster list
```
- 查看集群状态
```
obd cluster display myoceanbase
```
- 启动集群
```
obd cluster start myoceanbase
```
- 启动集群单个节点
```
obd cluster start myoceanbase --nodes 192.168.131.110
```
- 修改配置
```
obd cluster edit-config
```
