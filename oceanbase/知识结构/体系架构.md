# 物理结构
oceanbase 目录结构
|             |         |                                         |
| ----------- | ------- | --------------------------------------- |
| audit       |         | 存放审计日志                            |
| bin         |         | 存放 observer binary 文件               |
| etc         |         | 配置文件目录                                |
| etc2        |         | 配置文件额外保存目录（备份作用）        |
| etc3        |         | 配置文件额外保存目录（备份作用）        |
| lib         |         | 包目录                                  |
| log         |         | 运行日志目录                            |
| run         |         | 运行时目录，存放运行时产生的 PID 文件等 |
| store       | clog    | 动态数据写入的事务日志目录              |
|             | slog    | 静态数据写入的事务日志目录              |
|             | sstable | 数据目录                                |
